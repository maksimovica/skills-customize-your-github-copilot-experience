# 📘 Assignment: Lists and Dictionaries

## 🎯 Objective

Practice organizing related data in Python by using lists and dictionaries to build a simple student organizer.

## 📝 Tasks

### 🛠️ Create a Student Record

#### Description
Start by storing a few student records in a list of dictionaries.

#### Requirements
Completed program should:

- Create a list that stores at least three student records.
- Each record should include a name and a grade.
- Print the full list of students in a readable format.

### 🛠️ Add a New Student

#### Description
Write code that lets the program add a new student to the existing list.

#### Requirements
Completed program should:

- Accept a student name and grade from the user.
- Add the new information to the list of records.
- Display the updated list after the new student is added.

### 🛠️ Search for a Student

#### Description
Make it possible to look up a student by name.

#### Requirements
Completed program should:

- Ask the user for a student name.
- Search the list for a matching record.
- Print the student’s grade if found, or a friendly message if not found.
